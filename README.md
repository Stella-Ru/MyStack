# 🚀 MyStack

**My stack** 是一款专为高效能人士设计的个人规划 Web 应用。它摒弃了传统 Todo 应用的碎片化，采用了 **"宏观愿景 + 微观执行"** 的分层设计理念，结合清新的马卡龙配色与硬核的时间管理逻辑，帮助用户掌控全年节奏。

### ✨ 核心功能 (Key Features)

#### 1. 📅 多维视图系统 (Multi-View Architecture)
* **年视图 (Year Dashboard)**：独立的宏观仪表盘，包含年度挑战、核心目标、荣誉殿堂、备忘录四大板块，像日记本一样记录全年成就。
* **月/周分屏 (Split View)**：
    * **月视图**：可视化日历网格，支持跨天任务的**彩色条状显示**（带文字），直观展示任务跨度。
    * **周总览**：按周切割的独立目标管理，与日常任务解耦，专注于阶段性成果。
* **单日聚焦 (Daily Focus)**：左侧栏动态展示选中日期的任务清单，支持时间轴排序。

#### 2. ⚡️ 强力任务引擎 (Task Engine)
* **时间/日期一体化**：支持精确到分钟的 `datetime-local` 结束时间设置。
* **DDL 智能预警**：任务持续时间与截止日期 (DDL) 逻辑分离。配备**动态进度条**，根据剩余时间自动变色（绿 -> 黄 -> 红），提供直观的紧迫感。
* **马卡龙标签**：6 种低饱和度高级配色（樱花、泡芙、海盐等），兼顾分类与视觉美感。

#### 3. 📊 沉浸式复盘模式 (Review Mode HUD)
* **HUD 数据仪表盘**：开启后右下角弹出悬浮卡片，实时统计本月任务完成率、待办数与已完成数。
* **视觉降噪**：界面自动变暗，任务条转化为**极简符号**（✅ 完成 / ❌ 超时 / · 待办），帮助用户专注于评估执行情况。

#### 4. 🔒 隐私与体验 (Privacy & UX)
* **Local-First**：基于 `localStorage` 技术，所有数据均存储在用户本地浏览器，无需联网，数据绝对私密。
* **Aries Spark Design**：独家定制的 UI 图标与方正黑体排版，打造专业且舒适的视觉体验。

### 🛠 技术栈 (Tech Stack)
* React 18 + Vite
* Tailwind CSS (Styling)
* Lucide React (Icons)
* GitHub Pages / Vercel (Deployment)
